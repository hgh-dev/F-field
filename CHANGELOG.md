# Changelog

이 프로젝트의 모든 변경 사항은 이 파일에 기록됩니다.

이 형식은 [Keep a Changelog](https://keepachangelog.com/ko/1.0.0/)를 따릅니다.

## [1.3.1] - 2026-01-30
### 추가 (Added)
- **산림보호구역 레이어**: 산림보호구역(VWorld Data API)을 지도에 표시하는 오버레이 기능 추가.

### 변경 (Changed)
- **팝업 UI 및 편의성 개선**:
  - 주소(지번/도로명) 및 좌표 복사 시 복사된 항목을 명시하는 알림 메시지("~가 복사되었습니다") 표시.
  - '도로명', '좌표' 뱃지의 너비를 고정하여 정렬 개선.
- **보안 경고 강화**: 비공개 레이어 잠금 해제 성공 시 정보 유출 주의 문구 추가.

## [1.3.0] - 2026-01-29
### 추가 (Added)
- **지점 기록 기능**: 지도상의 특정 위치를 점(Marker) 형태로 저장하고 관리하는 기능 추가.
- **K-GeoP (스마트국토정보) 연동**: 팝업에서 버튼 클릭 시 주소를 복사하고 스마트국토정보 사이트로 연결하는 기능 추가.
- **캐시 방지 (Cache Busting)**: 배포 시 브라우저 캐시로 인해 업데이트가 반영되지 않는 문제를 해결하기 위해 스크립트 로딩 URL에 버전 파라미터(`?v=...`) 추가.

### 변경 (Changed)
- **팝업 UI 개선**: 
  - 버튼(지점기록, 지적기록, 공유, 길찾기) 스타일을 통일 (흰색 배경, 회색 테두리).
  - 버튼 배치 변경 (4개 버튼 1열 배치, 하단에 외부 서비스 링크 배치).
  - 주소 및 좌표 텍스트의 가독성 향상을 위해 줄 간격과 여백 조정.
- **토지e음/K-GeoP 버튼 스타일**: 활성화된 외부 연동 버튼을 파란색으로 강조.

### 수정 (Fixed)
- **토지e음 버튼 높이 문제**: 버튼 활성화 시 테두리 스타일 변경으로 인해 높이가 달라지는 문제 수정.
- **주소 복사 알림 중복**: 주소 복사 시 경고창이 두 번 뜨는 문제 해결.

## [1.2.0] - 2026-01-25
### 추가 (Added)
- **길찾기(네비게이션) 연동**: 티맵, 카카오맵, 네이버지도 앱으로 길찾기 연결 기능 추가.
- **비공개 레이어 암호화**: 특정 레이어 접근 시 암호를 입력받도록 보안 기능 추가.
- **주소 검색 기능 보완**: 검색 정확도 및 UI 개선.
- **토지e음 연동**: 선택한 지적의 PNU 코드를 이용해 토지e음(토지이용아카이브) 정보를 바로 조회하는 기능 추가.
