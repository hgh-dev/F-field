# 🌲 국유림 현장조사 앱 (F-Field) 사용자 가이드

**F-Field**는 현장에서 간편하게 지적(땅) 정보를 확인하고, 측량 및 기록을 남길 수 있도록 돕는 웹 애플리케이션입니다.

---

## 🚀 주요 기능

### 1. 🗺️ 지도 및 지적도 확인
- **배경 지도**: 위성지도, 지적도, 하이브리드 지도를 겹쳐서 볼 수 있습니다.
- **내 위치**: GPS를 이용하여 현재 내 위치를 지도상에 표시하고 추적합니다.
- **방위각 표시**: 이동 방향(나침반)을 지도에 표시하여 현재 바라보는 방향을 알 수 있습니다.

### 2. 🔍 주소 및 지번 검색
- 상단의 검색창을 통해 주소나 지번을 검색하여 해당 위치로 이동할 수 있습니다.
- 최근 검색 기록이 저장되어 빠르게 다시 찾을 수 있습니다.

### 3. 📝 현장 측량 및 기록
- **그리기 도구**: 하단 툴바를 이용해 지도 위에 직접 표시할 수 있습니다.
    - **점**: 특정 위치 표시
    - **선**: 거리 측정 (경로 그리기)
    - **면**: 면적 측정 (영역 표시)
- **지적 기록**: 지도상의 필지(땅)를 더블 클릭하면 빨간색으로 경계가 표시됩니다. 팝업의 **[지적 기록]** 버튼을 누르면 해당 영역이 내 기록에 저장됩니다.
- **지점 기록**: 팝업의 **[지점 기록]** 버튼을 누르면 해당 위치가 마커로 저장됩니다.
- **수정 및 삭제**: 저장된 기록을 터치하면 나오는 말풍선에서 바로 내용을 수정하거나 삭제할 수 있습니다.

### 4. 📂 기록 관리 (사이드바)
- 왼쪽 상단 메뉴(≡)를 누르면 **[기록 관리]** 탭이 나옵니다.
- 저장한 모든 측량/기록 데이터를 목록으로 볼 수 있습니다.
- **내보내기/불러오기**: 기록된 데이터를 파일(JSON)로 백업하거나 다른 기기로 옮길 수 있습니다.

### 5. 🔗 외부 공공 데이터 연동
- **토지e음**: 필지를 선택하고 **[토지e음 조회]** 버튼을 누르면 해당 토지의 규제 정보 등을 바로 확인할 수 있습니다.
- **K-GeoP (스마트국토정보)**: **[K-GeoP 조회]** 버튼을 누르면 주소가 복사되며, 스마트국토정보 사이트에서 상세 정보를 검색할 수 있습니다.

### 6. 🚗 편의 기능
- **길찾기**: 티맵, 카카오맵, 네이버지도 앱과 연동하여 해당 위치까지 바로 길 안내를 받을 수 있습니다.
- **위치 공유**: 현재 선택한 위치의 주소와 좌표를 문자가 카톡으로 공유할 수 있습니다.

---

## 💡 사용 팁

- **지적 경계 확인**: 지도의 빈 곳을 **더블 클릭**하면 그 위치의 지번 주소와 지적 경계가 빨간색으로 표시됩니다.
- **좌표 확인**: 상단의 주소창을 누르면 좌표 형식을 **도분초(DMS), 소수점(Decimal), TM좌표** 중에서 선택할 수 있습니다.
- **화면 캡처**: 측량 중이거나 수정 모드일 때는 화면 테두리에 **파란색 깜빡임 효과**가 나타나 현재 작업 중임을 알려줍니다.
