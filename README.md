# F-Field 사용자 가이드

**F-Field**는 현장에서 간편하게 지적(땅) 정보를 확인하고, 측량 및 기록을 남길 수 있도록 돕는 웹 애플리케이션입니다.
최신 위성지도와 지적도를 기반으로 현장 업무 효율을 극대화하세요.

---

## 주요 기능 및 사용법

### 1. 기본 UI
- **주소/좌표 표시** : 상단 바 중앙에 현재 위치의 주소와 좌표가 표시됩니다. 주소/좌표 표시줄을 터치하면 좌표 표시 형식을 변경할 수 있으며, 좌표 복사 버튼을 눌러 클립보드에 복사할 수 있습니다.
- **나의 위치 이동**: 우측의 첫번째 원형 버튼을 눌러 나의 위치로 이동할 수 있습니다.
- **나침반 모드**: 그 아래의 원형 버튼을 눌러 활성화하면 지도가 내 이동 경로를 따라갑니다.

### 2. 지도 설정 및 보기
사이드바 메뉴(좌측 상단 `≡` 버튼)를 통해 다양한 지도를 겹쳐볼 수 있습니다.

- **배경 지도 선택**:
    - **위성지도(VWorld)**: 국토교통부 VWorld에서 제공하는 안심 위성지도입니다.
    - **위성지도(Esri)**: 높은 해상도를 제공하는 Esri World Imagery 위성지도입니다. 산림 지역 확인에 유용합니다.
    - **일반지도**: 가독성이 좋은 기본 2D 지도입니다.

- **레이어 옵션**:
    - **하이브리드**: 주요 도로, 건물명, 지명이 표시됩니다. (지적도 아래에 표시되어 가독성이 좋습니다)
    - **행정경계**: 광역시도, 시군구, 읍면동, 리 경계를 한눈에 확인할 수 있습니다.
    - **지적도**:
        - **연속지적도**: 전산화된 지적도 및 임야도의 도면상 경계점을 연결하여 연속된 형태로 작성한 도면정보입니다.
        - **LX맵**: 전국 지자체 연속지적도를 지형에 부합하는 형태로 가공한 정보입니다.
    - **산림보호구역**: 산림보호구역 범위를 초록색 점선과 색상으로 표시합니다.

※ 모든 정보는 법적 효력이 없으며, 참고용으로만 활용하시기 바랍니다.

### 3. 강력한 검색 기능
상단 검색창을 통해 원하는 장소를 빠르고 정확하게 찾으세요.

- **주소/장소 검색**: 지번, 도로명 주소 또는 장소명을 입력하세요.
- **검색 결과**: 도로명 주소와 지번 주소가 구분되어 표시되며, 클릭 시 해당 위치로 부드럽게 이동합니다.
- **검색 기록**: 최근 검색한 장소가 자동 저장됩니다.

### 4. 현장 측량 및 기록 도구
지도 하단의 툴바를 이용해 현장 상황을 바로 기록하세요.

- **그리기 모드**:
    - **점(Marker)**: 특정 지점을 표시합니다.
    - **선(Polyline)**: 이동 경로를 그리거나 거리를 측정합니다.
    - **면(Polygon)**: 특정 구역을 표시하고 면적을 계산합니다.
- **시각 효과**: 측량이나 편집 중일 때는 화면 테두리에 **은은한 파란색 비네팅 효과**가 나타나 작업 중임을 직관적으로 알 수 있습니다.

### 5. 기록 관리 및 내보내기
- 사이드바의 **[기록 관리]** 탭에서 저장된 모든 데이터를 확인할 수 있습니다.
- **수정/삭제**: 지도 위의 기록을 터치하면 나오는 팝업에서 제목을 수정하거나 삭제할 수 있습니다.
- **데이터 백업**: 기록된 데이터를 `GeoJSON` 파일로 내보내거나 가져와서 다른 기기로 옮길 수 있습니다.

### 6. 지적 표시 및 외부 연동
- **지적 표시**: 지도를 **더블 클릭**하면 해당 필지의 경계가 빨간색으로 표시됩니다. 
- **위치 정보 표시**: 더블클릭한 필지의 **주소와 좌표 정보**가 말풍선 팝업으로 표시됩니다.
- **토지e음 연동**: 팝업 내 **[토지e음 조회]** 버튼을 눌러 토지이용계획 등의 정보를 바로 확인할 수 있습니다.
- **K-GeoP 연동**: **[K-GeoP 조회]** 버튼을 누르면 스마트국토정보 사이트에서 상세 정보를 검색하여 소유자 등의 정보를 확인할 수 있습니다.
- **길찾기**: 티맵, 카카오맵, 네이버지도 앱으로 연동하여 현재 위치까지 길 안내를 받을 수 있습니다.
- **위치 공유**: 현재 위치의 주소와 좌표를 문자나 카카오톡으로 동료에게 전송할 수 있습니다.
- **지점 기록**: 더블 클릭한 지점의 정보가 기록 관리 탭에 저장됩니다.
- **지적 기록**: 더블 클릭한 지적의 경계가 기록 관리 탭에 저장됩니다.

### 7. GPS 및 위치 추적
- **내 위치 찾기**: 우측 하단 조준경 아이콘을 누르면 현재 내 위치로 지도가 이동합니다.
- **위치 추적**: 나침반 아이콘을 누르면 지도가 내 이동 경로를 따라갑니다. (나침반 모드 시 방위각 표시)
